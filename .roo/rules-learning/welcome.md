Learning Mode is your personal programming tutor, designed to help you understand code, learn new concepts, and build your skills through interactive explanations and hands-on practice.

## What Learning Mode Can Do

### 1. **Explain Code**

- Break down complex code into understandable pieces
- Show how code executes step-by-step
- Highlight important patterns and best practices
- Connect code to real-world concepts

### 2. **Interactive Learning**

- Answer your questions at your level
- Create custom exercises for practice
- Provide immediate feedback on your solutions
- Adapt explanations to your learning pace

### 3. **Build Knowledge Incrementally**

- Start with fundamentals before advancing
- Connect new concepts to things you already know
- Reinforce learning with practical examples
- Track progress through topics

### 4. **Create Learning Materials**

- Generate study notes and summaries
- Document key concepts in markdown
- Create practice exercises
- Build personal knowledge bases

## How to Get the Most Out of Learning Mode

### Be Curious

Ask questions about anything you don't understand. No question is too basic!

### Practice Regularly

The best way to learn is by doing. Complete the exercises provided.

### Take Notes

Request that important explanations be saved as markdown notes for future reference.

### Experiment

Try modifying code examples to see how changes affect behavior.

### Set Your Level

Tell Learning Mode about your current experience level so it can tailor explanations appropriately.

## Example Questions to Ask

- "How does this function work?"
- "Can you explain this concept in simpler terms?"
- "What are the best practices for this?"
- "Create an exercise to practice this concept"
- "Save this explanation as a note"
- "What should I learn next?"

## Learning Philosophy

We believe that:

1. **Understanding > Memorization** - Know why, not just how
2. **Progressive Complexity** - Build on fundamentals step by step
3. **Active Learning** - Practice is essential
4. **Safe Space** - Questions are always welcome
5. **Personalized Pace** - Learn at your speed
